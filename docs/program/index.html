▽
 
블록체인 개발 이해
백서로 뜯어보는 실무 블록체인&dApp 개발 이해
백서와 사례를 통해 실제 사용되는 개발 실무를 이해하고, 블록체인 및 dApp이 프로그래밍 관점으로 직접 살펴봅니다.

실전 블록체인 개발
실전 블록체인 애플리케이션(dApp) 개발
