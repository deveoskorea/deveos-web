처음 시작하는 블록체인 개발 3주 입문


토큰을 발행하고 송금하는 스마트 컨트랙트를 직접 작성해보고 DAPP 서버에서 스마트 컨트랙트를 활용해볼 수 있는 블록체인 개발 입문 수업입니다.

효과적인 커리큘럼으로 블록체인 3주 입문


블록체인 개발을 위한 기초 문법 및 노하우


최신 블록체인 및 EOSIO 개발 기초의 이해
블록체인 개발 기초부터 실제 스마트 컨트랙트 구현까지 한 번에
최신 블록체인 중 하나인 EOS를 기반으로 하는 블록체인 애플리케이션 개발 입문 과정입니다. 3주동안 스마트 컨트랙트의 특징과 작성 방법을 알려드릴 것이며, dAPP 서버에서 스마트 컨트랙트를 어떻게 실행하는지 알려드립니다. 간단한 토큰 스마트 컨트랙트 예제를 통하여 토큰이 어떻게 발행, 송금되는지 배웁니다. 프로그래밍을 할 줄 몰라도 프로그래밍 언어를 배우는 수업이 아닌 블록체인의 구조 및 특징을 배우는 수업이기 때문에 비개발자 역시 손쉽게 따라오실 수 있습니다.
 
아직도 책으로만 쓰이지 않는 구세대 블록체인을 배우고 계셨나요? 블록체인 생태계는 현재 굉장히 빠른 속도로 발전하고 있어 책에 담지 못합니다. 제대로 된 로드맵이 없어서 혼자 하기엔 뭐부터 해야할지 막막하죠. 이제 현업 전문가에게 핵심적인 최신 내용들을 배워보세요!

코드를 읽는 방법부터 실전 프로그래밍까지, 단계마다 성취감을 직접 맛보며 블록체인 개발자로 입문할 수 있는 커리큘럼을 구성했습니다.
 
아래 중 하나라도 해당되시는 분들에게 "추천"합니다

블록체인 개발의 시작, Hello World!

EOS는 C++ 기반의 EOSIO-CPP로 스마트 컨트랙트 코드를 작성합니다. EOSIO-CPP, Hello World! 실습을 통해 EOSIO-CPP 프로그래밍에 필요한 문법 클래스, 함수와 스마트 컨트랙트 코드의 특징을 배웁니다. 테스트넷에서 스마트 컨트랙트 코드 작성부터 실행까지 실습으로 해보실 수 있습니다.

Token Smart Contract 구현하기

토큰이 어떻게 발행, 송금되는지 배웁니다. 간단한 token 스마트 컨트랙트 실습을 통하여 블록체인에 토큰이 어떻게 구현되어 있는가 배웁니다. 블록체인에 데이터가 어떤식으로 저장되는지 배우며, EOSIO Software, EOSIO CDT의 기능 및 특징을 배웁니다.

스마트 컨트랙트 API 활용하기

dAPP에서 스마트 컨트랙트를 실행하려면 서버에서 API 통신을 해야합니다. 이를 위해 API의 개념을 배운 후, 2주차에 작성한 token 스마트 컨트랙트를 기반하여 API를 만드는 실습을 진행합니다. 최신 트렌드 언어인 Python으로 EOS API 및 라이브러리를 활용하여 나만의 API를 만듭니다.
