최신 블록체인 및
EOSIO 개발 기초의 이해
백엔드부터 스마트 컨트렉트까지 한 번에
다양한 실습 예제들을 개발해보며 블록체인을 위한 C++, Python을 마스터합니다
Hello World는 프로그래밍의 시작입니다. [eosio-cpp, Hello World]는 C++ 기반 EOSIO 스마트 컨트렉트의 구조, C++ 필수 기초 지식 등을 하나씩 파고들어 블록체인 개발자의 길로 들어가는 입문 과정입니다. [Flask, Hello, World!] 로 백엔드의 핵심 업무인 Restful API의 개념부터 구현, 외부 API 연동까지 최신 트렌드인 Python의 Flask 프레임워크를 활용합니다.

코드를 읽는 방법부터 실전 프로그래밍까지, 단계마다 성취감을 직접 맛보며 블록체인 개발자가 될 수 있는 커리큘럼을 구성했습니다.
이 강좌, 바로 당신을 위해 준비했습니다
